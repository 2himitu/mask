# Teachable Machine를 활용해 마스크착용 여부를 인식 사이트
로그인 하여 이용 가능하며 Teachable Machine를 이용하여 
1. 마스크를 제대로 착용하였을 경우 
2. 마스크를 제대로 착용하지 안았을 경우 
3. 마스크를 착용하지 안 았을 경우
위 3개의 경우에 맞는 색과 음성으로 경고하도록 만들었다.

관리자의 계정으로 로그인시 위3개의 경우를 그래프로 집계된 정보를 볼수 있고 다른 회원을 관리
할수 있다.
